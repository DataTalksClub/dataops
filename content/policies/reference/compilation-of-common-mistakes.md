Compilation of Common Mistakes
LUMA
Mistake:

- Unable to add Alexey as the host and organizer of an event in Luma.

Process Document: [https://docs.google.com/document/d/1hR71aAIpAnbfW-YmW_Zp08wkKO0RFDCnms-i_2ctpho/edit?usp=sharing](https://docs.google.com/document/d/1hR71aAIpAnbfW-YmW_Zp08wkKO0RFDCnms-i_2ctpho/edit?usp=sharing)

- Wrong time of event in Luma announcement.

Process Document: [https://docs.google.com/document/d/1hR71aAIpAnbfW-YmW_Zp08wkKO0RFDCnms-i_2ctpho/edit?usp=sharing](https://docs.google.com/document/d/1hR71aAIpAnbfW-YmW_Zp08wkKO0RFDCnms-i_2ctpho/edit?usp=sharing)

How to Avoid:

- Create a checklist of everything that needs to be filled such as: Title, Subtitle, time and date, event banner, etc.

- Double-check everything before publishing the event.

ANNOUNCING EVENTS
Mistake:

- Not being able to announce an event in Slack

- Not being able to add the event to the Website

How to Avoid:

- Create a checklist on where every event should be published/ announced.

- Make use of Trello cards to be guided by the necessary actions to be done in every event.

Trello: [https://trello.com/b/qVB6fAUG/datatalksclub](https://trello.com/b/qVB6fAUG/datatalksclub)
