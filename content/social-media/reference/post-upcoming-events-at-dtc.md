# Post. Upcoming events at DTC

## Summary
- Purpose: Promote upcoming DataTalks.Club events on LinkedIn and Twitter/X.
- Outcome: Upcoming event carousel and thread copy are prepared for social posting.
- Trigger: The latest newsletter lists upcoming webinars, workshops, or podcasts to promote.
- Frequency: As needed when upcoming event roundups are posted.
## Prerequisites

- Access: Latest newsletter archive, Figma template, and social media accounts.
- Tools: Figma, PDF/image conversion, LinkedIn, and Twitter/X.
- Inputs: Upcoming event descriptions, speaker names, event links, and carousel visuals.
